# C++ 开源库精选基地

> 汇聚高效、强大的 C++ 开源库，助力您的项目开发

## 📚 项目简介

本项目是一个精选的 C++ 开源库集合，旨在为开发者提供高质量、经过实践验证的工具库和框架。涵盖了网络通信、并发编程、数据存储、日志系统、JSON 处理、脚本绑定等多个领域。

## 🌐 在线预览

项目主页：[C++ 开源库精选基地](https://levi-cpp-lib.github.io/)

## 🎯 涵盖领域

### 🌐 网络通信
- **ZeroMQ** - 高性能异步消息传输库
- **cppzmq** - ZeroMQ 的 C++ 绑定
- **gRPC** - 高性能 RPC 框架
- **cpp-httplib** - 轻量级 HTTP/HTTPS 库
- **cpr** - 现代化的 C++ HTTP 客户端
- **tacopie** - C++ 网络库
- **IXWebSocket** - WebSocket 客户端和服务器库
- **Eclipse Mosquitto** - MQTT 消息代理
- **libmodbus** - Modbus 协议库
- **open62541** - OPC UA 协议库

### ⚡ 并发与多线程
- **Cpp-TaskFlow** - 现代 C++ 并发编程库
- **SObjectizer** - Actor、发布/订阅和 CSP 框架
- **AOM** - Active Object Model
- **concurrentqueue** - 高性能无锁并发队列
- **asio** - 异步网络和 I/O 编程库
- **coost** - 现代 C++ 协程库

### 🗄️ 数据存储
- **redis-plus-plus** - 现代 C++ Redis 客户端
- **hiredis** - 轻量级 C 语言 Redis 客户端
- **leveldb** - 嵌入式键值存储库
- **SQLiteCpp** - SQLite 的 C++ 封装库

### 📝 日志系统
- **spdlog** - 快速、功能丰富的 C++ 日志库

### 📦 数据格式处理
- **nlohmann/json** - 流行的 C++ JSON 库
- **json11** - 轻量级 C++ JSON 解析库

### 🎨 脚本语言集成
- **sol2** - C++/Lua 绑定库
- **quickjs-ng** - 轻量级 JavaScript 引擎

### 🛠️ 工具库
- **args** - 命令行参数解析库
- **Folly** - Facebook 开源的 C++ 库集合
- **matplotplusplus** - 现代 C++ 图形库

## ✨ 特色功能

- 📋 **分类筛选** - 按编程语言（C++、Go、JavaScript、Lua、C）筛选库
- 📄 **分页展示** - 每页显示 6 个库，支持翻页导航
- 🎨 **精美 UI** - 现代科技风格设计，响应式布局
- 📝 **详细示例** - 每个库都有完整的使用示例和说明
- 🔗 **快速访问** - 一键跳转到官方文档和示例页面

## 🚀 快速开始

### 本地预览

1. 克隆项目仓库
```bash
git clone https://github.com/your-username/Common-open-source-projects.git
cd Common-open-source-projects
```

2. 使用浏览器打开 `docs/index.html` 即可查看主页

### 部署到 GitHub Pages

项目已配置 GitHub Pages，可直接访问：
```
https://levi-cpp-lib.github.io/
```

## 📖 使用指南

### 查看库列表

访问主页即可查看所有精选的 C++ 开源库，支持按语言筛选和分页浏览。

### 查看示例

点击每个库卡片上的「查看示例」按钮，即可跳转到详细的使用指南页面。

### 官方文档

点击「官方链接」按钮，可直接访问该库的官方网站或 GitHub 仓库。
